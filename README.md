# Customer Product Reviews Analysis

## Project Overview
This project analyzes customer product reviews using Python.

## Tasks Completed
1. Web Scraping
2. Exploratory Data Analysis (EDA)
3. Data Visualization

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- BeautifulSoup

## Files
- customer_reviews.csv
- web_scraping.py
- eda.py
- visualization.py

## Output
The project extracts customer reviews, performs analysis, and visualizes review patterns using charts and graphs.
